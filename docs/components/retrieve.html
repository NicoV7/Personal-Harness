retrieve — domain router + dual-scope merger + grep

Tier: Core · Files: src/server/retrieve/{index,router,grep}.ts · Last commit: f4f8801 (Wave 3)

Two-stage retrieval: domain-router YAML decides which sub-corpora to query for a given context; per-domain grep matches applies_when.paths + applies_when.symbols. Runs once over global, once over repo, merges with id-collision override. Ranks by severity × match × recency. Caps at 12 rules / 4 skills / 4 memories.
