auth — bearer middleware + DNS-rebinding guard

Tier: Edge · Files: src/server/auth/bearer.ts · Last commit: f4f8801 (Wave 3)

Hono middleware on every /mcp/* route. Reads token from BETTERAI_TOKEN_PATH (mode 0600). Verifies Authorization: Bearer <token>. Rejects requests with Host header outside 127.0.0.1:7777 / localhost:7777. /health is the only allowlisted path — bypass emits an audit event with source IP + UA.
