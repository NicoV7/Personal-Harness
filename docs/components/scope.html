scope — repo-root detection + dual-corpus discovery

Tier: Core · Files: src/server/scope/repo-detector.ts · Last commit: f4f8801 (Wave 3)

Given a context’s file_paths, walk up to the nearest .git/. If <repo-root>/.betterai/ exists, that’s the repo corpus. 60s mtime-keyed cache on .git/HEAD invalidates across branch switches. Output feeds retrieve + cache (key namespace) + audit (visibility).
