transport — Hono HTTP/SSE + MCP SDK adapter

Tier: Edge · Files: src/server/transport/http-sse.ts · Last commit: f4f8801 (Wave 3)

Hono app routes /mcp (POST) and /health (GET). The MCP SDK’s StreamableHttpServerTransport wraps the request/response. Bound to 127.0.0.1:7777 (localhost only). Stdio transport is never registered — permanently forbidden per BetterAI’s own repo rule.
