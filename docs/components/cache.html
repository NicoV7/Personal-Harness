cache — context_hash LRU + connection limiter

Tier: Core · Files: src/server/cache/{context-hash,connection-limiter}.ts · Last commit: f4f8801 (Wave 3)

Two collaborating mechanisms. context_hash LRU: 256 entries, 60s TTL, key includes scope + repo_root_detected (per the BetterAI repo-rule context-hash-includes-scope). Connection limiter: 16 in-flight retrieve_* calls; 64-deep queue beyond; HTTP 429 on overflow.
