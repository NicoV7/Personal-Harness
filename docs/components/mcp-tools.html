mcp-tools — 7 tool handlers

Tier: Tools · Files: src/mcp-tools/*.ts · Last commit: f4f8801 (Wave 3)

7 MCP tool handlers, one file each. retrieve_context is the canonical entry point (description starts with “ALWAYS call retrieve_context as your first action” per DX-FIX-8). Each handler validates input via Zod, calls the DI ToolContext, emits an audit event, returns shaped output.
